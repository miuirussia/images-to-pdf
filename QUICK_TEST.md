# Быстрый Тест (5 минут)

Минимальный набор проверок для подтверждения работоспособности:

## 1. Загрузка Изображений (30 сек)
- [ ] Drag & Drop 3 файла → добавились в список
- [ ] Кнопка "Выбрать файлы" → открылся диалог
- [ ] Превью отображаются

## 2. Управление Списком (30 сек)
- [ ] Перетащить первое изображение вниз → порядок изменился
- [ ] Удалить одно изображение → исчезло
- [ ] "Очистить всё" → список пуст

## 3. Создание PDF (1 мин)
- [ ] Загрузить 2-3 изображения
- [ ] Настройки: A4, Portrait, Fit
- [ ] Нажать "Создать PDF"
- [ ] Выбрать место сохранения
- [ ] Прогресс показан
- [ ] PDF создан и открывается

## 4. Responsive (30 сек)
- [ ] Уменьшить окно → настройки в Dialog
- [ ] Увеличить окно → настройки в боковой панели
- [ ] Прокрутка работает

## 5. Тема и Персистентность (1 мин)
- [ ] Переключить тему → изменилась
- [ ] Изменить настройки на Letter + Landscape
- [ ] Закрыть и открыть приложение
- [ ] Тема и настройки сохранились

## 6. Ошибки (30 сек)
- [ ] Перетащить .txt файл → Toast с ошибкой
- [ ] Без изображений → кнопка disabled

---

**Время:** ~5 минут
**Результат:** Если все ✅ — можно делать production build
